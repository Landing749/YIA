Standard Walking Procedures (SWP)

  • Eyes forward, no excessive talking.
  • Walk in pairs or solo. No YIA-identifying behavior.

Standard Classification System (SCS)

  • Code Blue: Suspicion/Shadowing Only
  • Code Yellow: Non-violent but concerning behavior
  • Code Red: Active threats or fights
  • Code Black: Bullying, harassment, coercion
